Book Typos

This page contains know typos for my books. If you spot a new one, please feel free to contact me so we can add it to the list.

The StatQuest Illustrated Guide to Machine Learning

  • Page 67, The graphs in the second box, for consistencies sake, should be labeled SSR(straight line) and SSR(parabola). However, it should be noted that in the wild world of statistics, SSR and RSS are often used interchangeably.
  • Page 81, Instead of the page being called “Logistic Regression: Weaknesses”, it should be called “Beyond Linear Regression”.
  • Page 102, Step3. The second line of the derivative of the SSR with respect to slope should be -2*2.3*(1.9 – (0 + 0.5*2.3)) instead of -2*1.9*(2.3 – (0 + 0.5*1.9)).
  • Page 135, The numerators for the two equations should be swapped.
  • Pages 138-145, I should have added a little note about the dataset that was used for this example. Some of it worked with logistic regression, some of it didn’t. Other parts worked with Naive Bayes and other parts didn’t – as a result, the sums are different.
  • Page 265, Step 2. The right hand side of the first equation should not be squared.
  • Page 275, Step 6. The final arrow should point to a slice of pumpkin pie instead of a slice of blueberry pie.
  • Page 293, Step 6. The -1 should be put inside parentheses.

The StatQuest Illustrated Guide to Neural Networks and AI

  • Pages 130-139, The bias right before the final ReLU should be 0.97 instead of 1.6, and the lower weight right after that ReLU should be 1.33 instead of -0.45

Typos for copies purchased before November 7th, 2025

  • Page 21, Step 7. The input for the neural network should be 1.0 instead of 0.4.
  • Page 35, Steps 3 and 5. The blue bent shape should be a blue straight line.
  • Page 49, Step 14. The step size should be 0.05 instead of 0.06
  • Page 54, Step 2. "First, we solve for the derivatives for each weight and bias with respect to the SSR" should be "First, we solve for the derivatives of the SSR with respect to each weight and bias."
  • Page 65, Step 16. Corrected a grammatical error by adding a “we” to the sentence.
  • Page 74, Step 11. Predicted_1 should be Predicted _2 and X_T1 should be X_T2.
  • Page 132, Step 6. The Dot Product should be 1 instead of -1.
  • Page 144-149 The first weight in the RNN (the one that comes right after the input) should be 1.1
  • Page 158, Step 6. 0.000000001 should be 0.000001.
  • Page 172, Step 5. The input to the LSTM should be 0.0 instead of 1.
  • Pages 233-234, Steps 28 and 29. One of the weights for the key for “go” is -1.4 but it should be -1.5.
  • Pages 233 and 235, Steps 27 and 31. The multiplication of the Values for “go” and the SoftMax values are swapped. NOTE: Since the SoftMax values are 0, nothing else changes.
  • Pages 265, Step 7, 270, Step 2, and 271, Step 1. The SoftMax output value for the input -1.2 should be 0 instead of 1.
  • Page 316, Step 8. The last equation should be (4 x -1) + (1 x 0) = -4.
